Les URL : Uniform Resource Locator

Qu'est-ce qu'une URL ?

Une URL est une adresse utilisée pour accéder à une ressource sur Internet. Elle se compose de plusieurs parties.

Exemple d'URL

Voici une URL : https://www1.ac-grenoble.fr/article/formations-scolaires-proposees-121615.

Parties d'une URL

Chaque partie de l'URL a une signification particulière, incluant le protocole, le nom de domaine et le chemin d'accès.
