Le CSS : Feuilles de style en cascade

Qu'est-ce que le CSS ?

CSS signifie Cascading Style Sheets. C'est un langage de description utilisé pour styliser les pages web.

Histoire du CSS

CSS a été créé pour améliorer la présentation des pages web, séparant le contenu de la mise en forme.

Utilisation du CSS

Le CSS permet de contrôler la mise en page, les couleurs et les polices des éléments HTML.
