Le HTML

1. Donner la signification de HTML ?

HTML signifie HyperText Markup Language.

2. Quel est l'histoire de ce langage ?

HTML a été créé par Tim Berners-Lee en 1991.

3. Est-ce un langage de programmation ou de description ?

HTML est un langage de description.

4. A quoi sert-il ?

HTML sert à structurer le contenu des pages web.

5. Quels sont les marqueurs de ce langage ?

Les marqueurs de HTML sont appelés balises.

6. Lister les principales balises, à quoi servent-elles ?

Les principales balises incluent : <html>, <head>, <body>, etc.

7. Aujourd'hui, l'usage de quelle version de HTML est recommandée par le W3C ?

HTML5 est la version recommandée par le W3C.
