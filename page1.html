Le HTML

Signification : HTML signifie HyperText Markup Language.

Histoire : Le HTML a été inventé par Tim Berners-Lee en 1991.

Langage de description : HTML est un langage de description et non de programmation.

Utilité : Il sert à structurer et afficher le contenu d'une page web.

Marqueurs : Les éléments HTML sont des balises comme <html>, <body>, <h1>, <p>, etc.

Balises principales : <html>, <head>, <body>, <p>, <h1>, <a>, <div>...

Version recommandée : HTML5 est la version recommandée aujourd'hui par le W3C.
