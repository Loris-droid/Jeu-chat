1. Donner la signification de HTML

HTML signifie HyperText Markup Language.

2. Quel est l'histoire de ce langage ?

HTML a été inventé par Tim Berners-Lee en 1991 pour structurer et présenter du contenu sur le web.

3. Est-ce un langage de programmation ou de description ?

HTML est un langage de description, utilisé pour structurer le contenu d'une page web.
