Le CSS

Signification : CSS signifie Cascading Style Sheets.

Histoire : CSS a été développé en 1996 par le W3C pour séparer le style du contenu HTML.

Langage de description : CSS est un langage de description utilisé pour le style des pages web.

Utilité : Il permet de styliser les pages HTML (couleurs, mises en page, etc.).

Sélecteurs : Les sélecteurs sont des moyens de sélectionner des éléments HTML (par balise, classe, identifiant).

Structure : CSS est structuré en règles : Sélecteur {propriété: valeur;}.

Version recommandée : CSS3 est la version actuelle recommandée par le W3C.
