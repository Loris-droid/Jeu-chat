Web Dynamique

Exemple : Une page de connexion ou un site e-commerce avec un panier interactif.

Utilité : Permet d'adapter le contenu selon les actions des utilisateurs.

Langages : JavaScript, PHP, Python sont utilisés pour rendre une page dynamique.
